六法JSON

六法JSONとは？

法律の条文を JSON 形式で取得できる Web API です。法律の条文を取得したい場合や、テストデータ等が欲しくなった際にご利用ください。現在利用は無料です。

ドキュメント

API ドキュメントは こちら からご確認いただけます。

開発状況

現在は発案者である OldBigBuddha が1人で開発・運用をしています。このサイトをいい感じにしてくれたり、法律データをいい感じにフォーマッティングするお手伝いをしてくれる人を募集しています。連絡は一番下の Twitter アカウントからお願いします。
